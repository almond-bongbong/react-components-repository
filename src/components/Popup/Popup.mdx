# Popup

- 최상위 컴포넌트 내에 `<PopupContainer />` 컴포넌트를 추가 합니다.
```
<div className="App">
  <PopupContainer />
</div>

```

## Basic usage

- 아래와 같은 형식으로 type을 통해 팝업을 활성화 합니다.
```
this.props.PopupActions.openPopup(popupTypes.SAMPLE)
```

<Playground>
  {() => {
    const TestWrap = connect(null, dispatch => ({
      PopupActions: bindActionCreators(popupActions, dispatch),
    }))(class Test extends React.Component {
      render() {
        return (
          <>
            <Button onClick={() => this.props.PopupActions.openPopup(popupTypes.SAMPLE)}>popup!</Button>
          </>
        )
      }
    })
    return <TestWrap />
  }}
</Playground>

## 데이터 전달

- 다음과 같이 데이터를 전달 할 수 있습니다.
```
this.props.PopupActions.openPopup({ type: popupTypes.SAMPLE, data: 'hello'})
```
