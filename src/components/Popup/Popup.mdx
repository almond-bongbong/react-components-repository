# Popup

- 최상위 컴포넌트 내에 `<Popup />` 컴포넌트를 추가 합니다.
```
<div className="App">
  <Popup />
</div>

```

- `<Popup />` 컴포넌트는 store의 `modal` reducer를 바라봅니다.
```
const initialState = {
  show: false,
  content: null,
};
```
