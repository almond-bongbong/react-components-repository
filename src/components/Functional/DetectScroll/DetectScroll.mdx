# DetectScroll

<PropsTable of={DetectScroll} />

- `endOffset` 속성을 이용해 어느 시점에서 end 이벤트를 발생 시킬지 설정할 수 있습니다. 기본 값은 0.9 이고 0 ~ 1 사이의 범위를 가집니다.

## Basic Usage

<Playground>
  scroll 👇
  <DetectScroll onScrollEnd={isEnd => console.log('end')} />
</Playground>
