# Alert

- 최상위 컴포넌트 내에 `<Alert />` 컴포넌트를 추가 합니다.
```
<div className="App">
  <Alert />
</div>

```

- `<Alert />` 컴포넌트는 store의 `alert` reducer를 바라봅니다.
```
const initialState = {
  show: false,
  message: null,
};
```
